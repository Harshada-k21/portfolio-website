Project Title: Personal Portfolio Website

Project Description:
This project is a personal portfolio website created to showcase my profile, technical skills, projects, and contact details. The website has a clean and modern design and is fully responsive for mobile, tablet, and desktop screens.

Features:

* Home section with introduction and typing animation
* About section with academic and career information
* Skills section displaying technical skills
* Services section showing capabilities
* Projects section with academic projects
* Testimonials section (UI feature)
* Contact form with basic validation
* Resume download option
* Smooth scrolling and fade-in animations
* Responsive design for all devices

Technologies Used:

* HTML5
* CSS3
* JavaScript
* Font Awesome (icons)
* Google Fonts (Poppins)
* Developed using VS Code

How to Run the Project:

2. Open the project folder.
